How can we access the element `'and'` in the array `['fish', 'and', 'chips']`?

MDN: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array

```javascript
let array = ['fish', 'and', 'chips'];
array[1]; // 'and'
```
